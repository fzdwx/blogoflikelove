# 二、快速启动

后端项目：

- 加载sql文件：\resources\db\myblogs.sql

- 修改配置文件为你对应的数据库连接地址

- 如果你没有部署经验，可以查看开发日志\项目部署.md

前端项目:

- npm install

- npm run serve
